Analiz edilen veri setinin hızlı görünümü

Kaggle.com üzerinden atanan veri seti 5077 satırdan ve 4 satırdan oluşmaktadır, Knime konsoluna geçmeden önce excel üzerinde analiz edildi. Bu analizle birlikte iki adet Class oluşturuldu bu classlar şu şekildedir;
1. Class : Avrupa Erkek
2. Class : Avrupa Kadın
3. Class : Asya Erkek
4. Class : Asya Kadın
5. Class : Amerika Erkek
6. Class : Amerika Kadın
1. Yaş grubu : 35-54
2. Yaş grubu : 55-74
3. Yaş grubu : 75+
Classların harf cinsinden olmasının Knime Araçalarından kimileri class elemanlarını sayı (integer) cinsinden istemekte, isteğe bağlı olarak knime üzerindeki strin to integer aracı da kullanılabilirdi. Veri Seti hakkında detaylı bilgi için menü barına göz atabilirsiniz!
